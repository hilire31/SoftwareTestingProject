@startuml

actor "Ship A" as A
actor "Ship B" as B
participant "Port Controller" as PC
participant "Dock Manager" as DM
participant "Crane Manager" as CM
participant "System Log" as LOG
actor "Operator" as OP

== Arrival Phase ==
A -> PC : Request dock & crane
B -> PC : Request dock & crane

== Resource Check ==
PC -> DM : Check dock availability
DM --> PC : Dock 1 and Dock 2 available
PC -> CM : Check crane availability
CM --> PC : Only Crane 1 available

PC -> A : Assign Dock 1 + Crane 1
PC -> B : Assign Dock 2 (wait for crane)
PC -> LOG : Record allocations

== Operation Phase ==
A -> D : Occupy Dock 1
A -> C : Use Crane 1 for loading
A -> LOG : Operation started

== Completion ==
A -> PC : Operation completed
PC -> DM : Release Dock 1
PC -> CM : Release Crane 1
PC -> LOG : Record release

== Next Allocation ==
PC -> B : Assign Crane 1
B -> D : Occupy Dock 2
B -> C : Use Crane 1 for loading
B -> LOG : Operation started

== Monitoring ==
OP -> LOG : View port status and resource usage

@enduml
